# 03 卷积到底在做什么

## 1. 先用一个小图找竖线

输入图像是 4 行 4 列：

```text
1 0 2 1
0 1 3 1
2 1 0 0
1 2 1 0
```

现在拿一个 2x2 小模板去看左上角这一块：

```text
输入块        卷积核
1 0           1 0
0 1           0 1
```

对应相乘再相加：

```text
1*1 + 0*0 + 0*0 + 1*1 = 2
```

这个 2 就是输出特征图左上角的一个值。卷积最基本的动作就是：拿一个小窗口覆盖输入的一小块，和卷积核逐项相乘，再把所有乘积加起来。

## 2. 直觉：卷积核是一把会滑动的尺子

卷积核可以理解成一个小尺子。它不是只量一个位置，而是在整张图上滑动。

如果卷积核喜欢某种局部模式，例如竖线、横线、亮暗变化，那么它滑到相似位置时输出值会大，滑到不相似位置时输出值会小。

卷积真正有用的地方在于：

- 它看局部区域，而不是一次看完整张图。
- 同一组权重在不同位置反复使用。
- 输出仍然保留空间位置关系。

## 3. 正式定义

对单通道输入，设输入为 `I`，卷积核为 `K`，输出为 `O`。不考虑 padding，stride 为 1 时：

```text
O[oh, ow] =
  sum over kh, kw of I[oh + kh, ow + kw] * K[kh, kw]
```

其中：

- `oh, ow` 是输出位置。
- `kh, kw` 是卷积核内部位置。
- `sum` 表示把窗口内所有乘积累加成一个输出值。

## 4. 完整手算：算出一整张输出

输入：

```text
1 0 2
0 1 3
2 1 0
```

卷积核：

```text
1 0
0 1
```

输入是 3x3，卷积核是 2x2，stride 为 1，不 padding，所以输出是 2x2。

先算输出左上角 `O[0,0]`，窗口是：

```text
1 0
0 1
```

手算：

```text
1*1 + 0*0 + 0*0 + 1*1 = 2
```

再算输出右上角 `O[0,1]`，窗口向右滑一格：

```text
0 2
1 3
```

手算：

```text
0*1 + 2*0 + 1*0 + 3*1 = 3
```

再算输出左下角 `O[1,0]`，窗口向下滑一格：

```text
0 1
2 1
```

手算：

```text
0*1 + 1*0 + 2*0 + 1*1 = 1
```

最后算输出右下角 `O[1,1]`：

```text
1 3
1 0
```

手算：

```text
1*1 + 3*0 + 1*0 + 0*1 = 1
```

所以输出特征图是：

```text
2 3
1 1
```

这就是读者必须先掌握的卷积本体：一个输出值来自一个窗口和一个卷积核的逐项乘加。

## 5. 多通道时发生了什么

如果输入有 3 个通道，卷积核也要有 3 个对应通道。一个输出值不再只累加 2x2，而是累加：

```text
卷积核高 * 卷积核宽 * 输入通道数
```

例如 2x2 卷积核、3 个输入通道，就是：

```text
2 * 2 * 3 = 12 次乘法，再把 12 个乘积加起来
```

多个输出通道意味着有多套卷积核。每套卷积核生成一个输出通道。

## 6. 图解：窗口滑动

```text
输入 3x3                 输出 2x2

[1 0] 2                  O00 O01
[0 1] 3                  O10 O11
 2 1  0
  |
  +-- 当前窗口算 O00

窗口向右：

1 [0 2]                  O01 使用右上窗口
0 [1 3]
2  1 0

窗口向下：

1  0 2
[0 1] 3                  O10 使用左下窗口
[2 1] 0
```

## 7. 连接到 NPU

现在把卷积放到 NPU 视角看：

- 一个输出值是多次 MAC 的累加。
- 不同输出位置可以并行计算。
- 同一个输入像素会被相邻窗口重复使用。
- 同一个卷积核会在整张图上重复使用。
- 多个输出通道可以使用多套权重并行生成。

这说明卷积天然适合 NPU：它有大量规则重复的乘加，也有明显的数据复用机会。但它也给 NPU 带来压力：输入窗口、权重、部分和都要被安排在合适的片上存储和阵列位置，否则算力会等数据。

## 8. 常见误区

- 误区一：卷积核越大越高级。卷积核大意味着每个输出要做更多乘加，也可能更难复用。
- 误区二：卷积只是在“压缩图片”。卷积也可以增加通道、提取特征、改变空间尺寸。
- 误区三：输出值是凭空学出来的。每个输出值都可以展开成具体的乘加式。
- 误区四：多通道只是多张图。多通道会让每个输出值的归约长度变长。
