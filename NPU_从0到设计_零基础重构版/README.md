# NPU 从 0 到设计：零基础重构版

这套笔记的目标不是先背 NPU 名词，而是从能手算的小例子开始，逐步把“数字怎样变成向量和矩阵”“图像怎样变成特征图”“卷积怎样变成多重循环”讲清楚。

## 第一批阅读顺序

1. [00_怎么学习NPU.md](00_怎么学习NPU.md)
2. [01_数字_向量_矩阵_乘加.md](01_数字_向量_矩阵_乘加.md)
3. [02_图像_特征图_通道.md](02_图像_特征图_通道.md)
4. [03_卷积到底在做什么.md](03_卷积到底在做什么.md)
5. [04_卷积如何变成循环.md](04_卷积如何变成循环.md)

## 学习约定

- 每章先看具体数字，再看定义。
- 每章都要至少手算一次，不跳过中间过程。
- 看到 NPU 名词时，优先问三个问题：它在算什么、数据从哪里来、结果放到哪里去。
- Git 仓库和思源笔记都是发布目标；本目录是唯一作者源。
